# 🔊 Text-to-Speech Converter

## 📌 Overview

A Python tool that converts text into speech using a text-to-speech engine.

## 🚀 Features

* Converts text to audio
* Simple and fast execution

## 🛠️ Tech Used

* Python
* pyttsx3

## ▶️ Run

```bash
python main.py
```

## ⚠️ Note

Ensure required libraries are installed.
